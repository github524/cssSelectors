CSS Selectors

CSS Code Explanation
p a Style the child (a) of p
p, a Style p and a
p > a Style direct descendent of p
p + a Style the a next to p (just after)
p ~ a Selects ALL a's after the p
::before / ::after Add content from CSS before or after element
a:nth-child(even) Select every other a
a:first-child Select 1st element inside a
a[href*="#"] Select all a elements that have a # in their href
a:not(.name) Select all a that do not have class of "name"
h1.large Select all h1 that have a class of "large"
